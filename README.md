# Duna Architects | Next.js

A Next.js conversion of the Duna Architects cinematic landing page.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - App Router pages and layout
- `components/` - React components (Navbar, HeroSection, VisionSection, etc.)
- `app/globals.css` - Global styles and Tailwind

## Features

- **Scroll-driven hero** - Arch zooms and expands as you scroll
- **Reveal animations** - Sections fade in on scroll via Intersection Observer
- **Horizontal project carousel** - Selected works with smooth scroll
- **Responsive design** - Mobile-first with Tailwind breakpoints
- **Custom theme** - Cormorant Garamond + Montserrat, eggshell palette

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run start` - Start production server
